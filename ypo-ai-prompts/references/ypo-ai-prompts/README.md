## Folder Map

- `output/category-index.csv` - spreadsheet-friendly list of every category, count, and file name.
- `output/category-index.md` - readable index of every category.
- `output/categories/` - one Markdown file per category.

## Note

A prompt may appear in more than one category file because YPO assigns multiple category tags to most prompts. This is intentional, like cross-listing a video in multiple playlists.
